# Decision: Project Charter v1.0

Date: 2026-06-30

## Decision

Cindy Tennis Lab adopts `PROJECT-CHARTER.md` as the formal Day 0 to Day 2
foundation charter.

The charter defines the project as a long-term Learning Operating System
exploration, with tennis as the first experiment field.

## Context

The project started from a practical question:

> AI 能否帮助一个普通家庭，更科学地学习一项复杂技能？

The first concrete experiments are:

- Mickey: adult tennis learner, USTA/NTRP 2.5 to 4.0
- Cindy: junior tennis learner, long-term NCAA pathway

The project has moved beyond scattered conversation. Future work should be
versioned in the repository through research, RFC, review, commits, and
milestones.

## Consequences

The repository should treat these as durable assets:

- Knowledge
- Data
- Experiment
- Time

The next priority is Tennis Knowledge Graph v0.1.

Serve should be modeled as a `Skill` domain within the knowledge graph, not as
the top-level organizing principle.

The local Git tag `v0.1-foundation` should mark the commit that contains the
charter.

Future Agent work should wait until the knowledge and data foundation is stable
enough to support it.
