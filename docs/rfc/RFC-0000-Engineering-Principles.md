# RFC-0000: Engineering Principles

Status: Accepted

Date: 2026-06-30

## Summary

This RFC defines how Cindy Tennis Lab should work as a research and engineering
project.

## Principles

- Knowledge First
- Data First
- Evidence First
- Version Everything
- Small Commits
- Long-term Thinking

## Workflow

```text
Issue
↓
Research
↓
RFC
↓
Review
↓
Codex
↓
Commit
↓
Merge
```

## Rules

1. Start with a clear issue.
2. Write or update an RFC for durable decisions.
3. Keep claims tied to evidence.
4. Keep commits small and reviewable.
5. Do not build Agents before the knowledge foundation is ready.

## Non-Goals

This RFC does not define any Agent, app, database, video pipeline, or UI.
