# Mickey Experiments

Mickey is an adult tennis learner with a long-term goal of moving from 2.5
toward 4.0.

The purpose of this space is to run small, evidence-driven learning experiments:

- understand the game before copying shapes
- connect practice notes to match outcomes
- use video and reflection to find repeated patterns
- build technique, footwork, tactics, and mental habits through iteration

## Current Direction

Understanding is more important than imitation.

The 2.5 to 4.0 path should be built through first principles, evidence-driven
practice, engineering thinking, continuous iteration, and long-termism.

AI does not replace learning. AI amplifies learning.
