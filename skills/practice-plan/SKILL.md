# Practice Plan

Turn one focus into a practical session.

## Inputs

- learner
- session length
- current focus
- available equipment
- physical constraints
- recent evidence

## Workflow

1. State the session goal in one sentence.
2. Choose warm-up, main work, constraint drill, and review.
3. Keep volume age-appropriate and health-aware.
4. Define what evidence will be captured.
5. End with one reflection question.

## Output Format

```markdown
## Session Goal

## Warm-Up

## Main Work

## Constraint Drill

## Evidence To Capture

## Stop Conditions

## Reflection Question
```

## Guardrails

- Do not create high-load plans when pain or injury is present.
- Do not add more than one main technical focus.
- For children, preserve joy and confidence.
