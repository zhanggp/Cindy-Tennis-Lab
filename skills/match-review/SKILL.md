# Match Review

Review matches to learn, not to assign blame.

## Inputs

- learner
- match date
- opponent or level
- score
- point patterns
- emotional notes
- serve and return notes

## Workflow

1. Summarize the match context.
2. Identify repeated patterns.
3. Separate controllable from uncontrollable factors.
4. Connect match patterns to practice priorities.
5. Choose one next training priority.

## Output Format

```markdown
## Match Summary

## Patterns

## Serve And Return

## Pressure Moments

## Controllables

## Next Training Priority
```

## Guardrails

- Do not reduce the match to win or loss.
- Do not infer character from performance.
- Do not recommend many changes at once.
