# Skills

Skills are reusable AI workflows for Cindy Tennis Lab.

They help turn recurring work into repeatable structure: review a serve video,
write a monthly report, create a practice plan, review a match, or capture new
knowledge.

## First Skills

- [serve-video-review](serve-video-review/SKILL.md): review a short serve clip.
- [monthly-growth-report](monthly-growth-report/SKILL.md): summarize a month of learner evidence.
- [practice-plan](practice-plan/SKILL.md): turn one focus into an actionable practice session.
- [match-review](match-review/SKILL.md): convert match evidence into patterns and next work.
- [knowledge-capture](knowledge-capture/SKILL.md): promote observations into structured knowledge.

## Principle

AI does not replace learning. AI amplifies learning.

Each skill should preserve three things:

- evidence before opinion
- one clear next action
- long-term learner health
