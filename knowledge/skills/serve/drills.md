# Serve Drills

Drills are practice designs that test one part of the skill.

This file defines drill categories only. It is not a daily training plan.

## Drill Categories

| Category | Purpose | Primary Metric |
| --- | --- | --- |
| Toss window | Build repeatable ball placement | Toss in target window |
| Trophy shape | Build stable side-on model | Shape score |
| Smooth drive | Link legs, trunk, and arm | Rhythm score |
| Contact height | Build upward reach and margin | Contact score |
| Landing balance | Protect control and recovery | Balanced landing count |
| Serve-plus-one | Connect serve to point start | Ready recovery count |

## Drill Rule

Each drill should define:

- one focus
- one constraint
- one metric
- one stop condition
- one evidence artifact

## Stop Conditions

Stop or downgrade the drill if:

- pain exceeds the agreed threshold
- form changes because of fatigue
- the player loses confidence or clarity
- the drill starts training a compensation pattern
