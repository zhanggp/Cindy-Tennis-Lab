# Serve Motor Learning

Motor Learning describes how the serve becomes repeatable and transferable.

## v0.1 Learning Rule

One serve cycle should focus on one primary change.

For Cindy, do not change toss, stance, trophy, leg drive, contact, and follow
through all at once.

## Learning Progression

| Stage | Goal | Evidence |
| --- | --- | --- |
| Awareness | Player can name the focus | Reflection, coach note |
| Shape | Player can show the action slowly | Shadow motion, still frame |
| Repetition | Player can repeat in low pressure | Practice count |
| Transfer | Player can use it in point context | Serve-plus-one or match clip |
| Review | Player and coach can compare before/after | Monthly evidence |
