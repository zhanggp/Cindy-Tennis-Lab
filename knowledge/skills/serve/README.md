# Serve

Serve is the first concrete Skill node in Cindy Tennis Lab.

In the Tennis Knowledge Graph, Serve is not a top-level domain. It is a Skill
that connects Physics, Biomechanics, Anatomy, Motor Learning, Drill, Metrics,
and Experiment.

## Purpose

Serve Ontology v0.1 defines how the project models the tennis serve before
building training plans, video analysis, or Agents.

## Core Principle

For early development, serve quality is not speed first.

The first goal is a stable action model:

- stable toss
- side-on body shape
- smooth upward drive
- relaxed swing
- balanced landing
- ready recovery
