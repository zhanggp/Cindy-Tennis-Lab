# Engineering Principles

Cindy Tennis Lab is a research and engineering project.

The project should move through small, evidence-backed steps.

## Principles

### Knowledge First

Build durable knowledge before building Agents, apps, or automation.

### Data First

Evidence, observations, metrics, and experiments are first-class project
assets.

### Evidence First

Every claim should cite evidence when evidence exists.

### Version Everything

Important decisions, schemas, reports, and experiments belong in Git.

### Small Commits

Prefer one goal, one output, one commit.

### Long-Term Thinking

Design choices should still make sense years from now.
