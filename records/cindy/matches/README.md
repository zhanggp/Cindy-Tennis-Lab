# Cindy Match Records

This directory is for public-safe match templates and anonymized summaries only.

Do not commit:

- real event dates or locations
- opponent names
- private coach notes
- medical, pain, or sensitive personal details
- identifiable video links

Real match baselines should live under `data/private/` or `profiles/private/`.

Use the public template at:

- `experiments/templates/match-baseline.example.yaml`
