# AI Feedback Template

Use this template when asking AI to review a short tennis clip.

## Prompt Structure

Review this clip as a learning companion.

Focus on:

- observable evidence
- one main strength
- one main issue
- one next practice action

Do not overload the learner with too many corrections.
