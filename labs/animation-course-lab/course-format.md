# Course Format

Each lesson should combine:

1. Professional player slow motion: why it works.
2. 3D skeleton animation: how the body sequence works.
3. Cindy's version: what is happening now.
4. Overlay comparison: what differs.
5. Home practice: what to do next.

## Lesson Length

Aim for 5 minutes for child-facing lessons.

## Feedback Rule

One lesson should teach one primary idea.
