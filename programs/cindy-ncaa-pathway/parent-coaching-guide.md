# Parent Coaching Guide

The parent is not replacing the coach.

The parent protects the learning environment.

## Focus

- Ask better questions.
- Record evidence consistently.
- Praise effort, courage, and attention.
- Avoid changing too many things at once.
- Help Cindy understand why she practices something.

## Feedback Rule

Give one clear focus at a time.

Too much feedback creates noise. A small useful cue can create progress.
