# Roadmap

This roadmap keeps Cindy Tennis Lab small enough to build and large enough to
matter.

## v0.1 Foundation

- Define the mission, vision, manifesto, and why.
- Create the first system architecture overview.
- Establish first principles for tennis learning.
- Create the first tennis knowledge schema.
- Create separate experiment spaces for Mickey and Cindy.
- Start a daily journal habit.

## v0.2 Knowledge Capture

- Add templates for practice notes, match notes, video review, and coaching
  observations.
- Capture recurring concepts such as balance, contact point, spacing, timing,
  recovery, serve patterns, rally tolerance, and decision making.
- Connect observations to evidence and next actions.

## v0.3 AI-Assisted Review

- Explore AI workflows for summarizing practice sessions.
- Explore video annotation pipelines.
- Build prompts and evaluation rubrics for tennis learning feedback.
- Compare AI suggestions against coach input and real outcomes.

## v0.4 Player Pathways

- Build Mickey's 2.5 to 4.0 adult learner pathway.
- Build Cindy's junior development and NCAA pathway map.
- Track technical, tactical, physical, mental, and academic dimensions.

## v1.0 Learning Operating System

- Create a coherent tennis learning operating system that combines knowledge,
  evidence, experiments, review, and long-term planning.
- Keep the core principle intact: Build with AI. Grow together.
